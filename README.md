# myconfig

Personal configuration files.

## Ghostty

Terminal emulator configuration for [Ghostty](https://ghostty.org/).

Key features of the config:
- Theme: Catppuccin (light/dark auto-switch)
- Font: Maple Mono NF CN
- Background transparency with blur effect
- Quick terminal toggle: `Ctrl + grave_accent`
- macOS Option key as Alt
